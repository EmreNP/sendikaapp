# Logo Dosyası

Lütfen eklediğiniz "TÜRK DİYANET VAKIF-SEN KONYA" logosunu bu klasöre `tdv-sen-logo.png` olarak kaydedin.

**Logo hazırlama adımları:**
1. Eklediğiniz circular logo görselini açın
2. Arka planı transparent yapın (veya sadece yuvarlak iç kısmı kesin)
3. PNG formatında kaydedin
4. Dosya adı: `tdv-sen-logo.png`
5. Bu klasöre (`mobile/assets/`) kopyalayın

Alternatif: Yuvarlak logo içindeki merkez kısmı (Mevlana Müzesi görseli + yıldız) kırparak da kullanabilirsiniz.
