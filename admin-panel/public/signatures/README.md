# İmza Dosyaları

Bu klasöre PDF formlarında varsayılan olarak kullanılacak imza görsellerini koyun.

## Dosyalar

| Dosya adı              | Hangi alan         | Açıklama                        |
|------------------------|--------------------|---------------------------------|
| `basknsignature.png`   | `basknsignature`   | Başkan imzası (kayıt formunda)  |

## Nasıl değiştirilir?

1. İmzayı PNG veya JPEG olarak hazırlayın (şeffaf arka plan önerilir).
2. `basknsignature.png` adıyla bu klasöre kaydedin.
3. Sayfayı yenileyin — önizleme açıldığında alan otomatik dolu gelir.

> Not: Dosya yoksa alan boş açılır, admin el ile imza ekleyebilir.
